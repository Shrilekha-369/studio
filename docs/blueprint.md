# **App Name**: One Fitness Studio

## Core Features:

- Demo Booking: Allow users to view the class schedule and book slots.
- Gallery: Showcase the competitions won and venue as well.
- Contact us: Provides information about One Fitness Studio (include a maps API to click)

## Style Guidelines:

- Primary color: Vibrant orange (#F26419), reflecting energy and dynamism, taken directly from the 'ONE Fitness Studio' logo.
- Background color: Light, desaturated orange (#FAE8DD), a softer tone of the primary to create a welcoming yet focused environment.
- Accent color: A slightly red-tinted orange (#F25430) adds contrast and calls attention to interactive elements.
- Font Pairing: 'Poppins' (sans-serif) for headings to keep it modern, 'PT Sans' (sans-serif) for body.
- Use modern, clean icons related to fitness activities.
- Maintain a clean and intuitive layout, prioritizing user experience and easy navigation.
- Subtle animations to provide feedback and enhance engagement without overwhelming the user.
